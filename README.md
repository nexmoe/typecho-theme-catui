## 纯CSS显示博主

```CSS
#ds-thread #ds-reset a.ds-user-name[data-user-id='这里是你的多说id']:after {
content: "博主";
margin-left: 6px;
font-size: 12px;
color: #13BF20;
background: rgba(255, 255, 255, .35);
border-radius: 4px;
padding: 0 3px;
}
```
